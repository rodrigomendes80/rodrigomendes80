# 👩🏻‍💻 Rodrigo Mendes

**`Graduando em Ciências de Dados`**

Me chamo Rodrigo Mendes de Oliveira, tenho 45 anos e sou natural de São Paulo. Minha primeira formação é em Análise de Sistemas, onde atuei com desenvolvimento de aplicações web e banco de dados relacional. Atualmente, estou cursando Ciência de Dados pela UNIVESP e trabalho com consultoria de Marketing Digital para empresas, unindo tecnologia, dados e estratégias de crescimento.

Ao longo da minha trajetória, percebi que a análise de dados é um pilar fundamental para potencializar campanhas de marketing, trazendo insights valiosos que permitem otimizar investimentos, compreender o comportamento do consumidor e aumentar a performance das ações digitais. Minha missão é ajudar empresas a tomarem decisões mais inteligentes e baseadas em dados, transformando informações em resultados reais.
